# CC123 Java Project 1
This repo is a compilation of all the solutions to the given problem from CC123 File.<br>

# Usage 
1. Open your favorite Java IDE.<br>
2. Run `Main.java`<br>

# Command Line Usage
Copy the command:<br>
```
git clone https://github.com/wrkean/java-project1-cc123.git
javac Main.java
```
Run:<br>
```
java Main
```
